Tetris Game in Java


This is a classic Tetris game implemented in Java, utilizing Swing for the graphical user interface. The game offers intuitive controls and engaging gameplay, where players can manipulate falling blocks to create solid rows and clear them from the screen.

Features:
- Classic Tetris gameplay experience.
- User-friendly interface with simple controls.
- Smooth animations and graphics.
- Title screen with instructions and a play button.
- Automatic game start upon clicking the play button.
- Responsive design to adapt to different screen sizes.


Installation:
- Ensure you have Java Development Kit (JDK) installed on your system.
- Download or clone the repository to your local machine.
- Open the project in your preferred Java IDE.
- Compile and run the Window.java file.


How to Play:
Upon launching the game, you'll be greeted with the title screen.Read the instructions displayed on the screen. Click on the "Play" button to start the game. Use the arrow keys to move the falling Tetris blocks left or right. Press the down arrow key to accelerate the block's descent. Rotate the block using the up arrow key.Your goal is to create solid horizontal rows with the falling blocks. Completed rows will disappear, earning you points. If the stack of blocks reaches the top of the screen, the game ends.

Controls:
- Left Arrow Key: Move block left.
- Right Arrow Key: Move block right.
- Down Arrow Key: Accelerate block descent.
- Up Arrow Key: Rotate block.
